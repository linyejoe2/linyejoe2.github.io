筆記

這題要訓練 stack 的運用，解題步驟如下：

  1. 建立一個 stack ，運用他 LIFO 的特性，我們可以很好的對剛插入的值進行比較。
  2. 遍歷 s
  3. 遇到左括弧( '(', '[', '{' )，都塞進去 stack
  4. 遇到右括弧( ')', ']', '}' )，就檢查 stack 最後一個值，
    • 如果一樣，就把這個值刪掉( pop() )
    • 如果不一樣，代表這個字串不合法，回傳 false
  5. 最後檢查 stack 裡還有沒有值，有就傳 false ，沒有就傳 true

Java iterative solution.
